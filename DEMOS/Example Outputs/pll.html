🔁 PLL — Key Components

A Phase-Locked Loop is a feedback control system that locks the output phase to a reference signal. Here's what each block does:

1. 📡 Phase Detector (PD)

  • Compares the phase of the input reference θref with the fed-back output phase θout
  • Produces an error signal proportional to the phase difference:

e(t)=θref(t)−θout(t)

2. 🔽 Loop Filter (LF)

  • Typically a low-pass filter — smooths the error signal and removes high-frequency noise
  • Determines the loop bandwidth and stability
  • Common form (first-order):

F(s)=1+sτ21+sτ1

3. 🔊 Voltage-Controlled Oscillator (VCO)

  • Converts the filtered voltage into a frequency:

ωout(t)=ω0+KVCO⋅v(t)

  • Integration of frequency gives phase:

θout(s)=KVCOs⋅V(s)

4. ➗ Frequency Divider (optional)

  • In frequency synthesis, divides output by N to multiply the reference frequency:

fout=N⋅fref

📐 PLL Open-Loop Transfer Function

G(s)=KPD⋅F(s)⋅KVCOs

The 1s from the VCO makes the PLL a Type 1 system — it has zero steady-state phase error to a constant phase input.

🔒 Closed-Loop Transfer Function

H(s)=G(s)1+G(s)=KPD⋅F(s)⋅KVCOs+KPD⋅F(s)⋅KVCO
