Linear Quadratic Gaussian (LQG) Control Design

A Second-Order Position-Velocity System

1. State-Space Model

For a mass subject to force input with process noise:

$$\dot{\mathbf{x}} = A\mathbf{x} + Bu + \mathbf{w}$$ $$\mathbf{y} = C\mathbf{x} + \mathbf{v}$$

States: $\mathbf{x} = \begin{bmatrix} p \\ v \end{bmatrix}$ — position and velocity

System matrices:

$$A = \begin{bmatrix} 0 & 1 \\ 0 & 0 \end{bmatrix}, \quad B = \begin{bmatrix} 0 \\ \frac{1}{m} \end{bmatrix}, \quad C = \begin{bmatrix} 1 & 0 \end{bmatrix}$$

A double integrator: velocity integrates to position, force integrates to velocity.

2. LQR Cost Function

Minimise the infinite-horizon quadratic cost:

$$J = \int_0^\infty \left( \mathbf{x}^T Q \mathbf{x} + u^T R\, u \right) dt$$

Tuning matrices:

$$Q = \begin{bmatrix} q_p & 0 \\ 0 & q_v \end{bmatrix}, \quad R = [r]$$

  • $Q$ penalises state deviation (position error, velocity error)
  • $R$ penalises control effort (force applied)
  • Large $q_p$ → tighter position tracking
  • Large $r$ → gentler, energy-conservative control

3. Algebraic Riccati Equation (ARE) — LQR

The optimal gain $K$ is found by solving the continuous-time ARE:

$$A^T P + P A - P B R^{-1} B^T P + Q = 0$$

where $P = P^T \succ 0$ is the unique positive-definite solution.

Optimal state feedback gain:

$$\boxed{K = R^{-1} B^T P}$$

Closed-loop dynamics (full state feedback):

$$\dot{\mathbf{x}} = (A - BK)\mathbf{x} + BK\mathbf{x}_{\text{ref}}$$

The eigenvalues of $(A - BK)$ are placed optimally in the left-half plane.

4. Kalman Filter (LQE) — Observer Design

Since full state is not directly measured, we estimate $\hat{\mathbf{x}}$ from noisy output $\mathbf{y}$.

Noise assumptions:

$$\mathbf{w} \sim \mathcal{N}(0,\, Q_n), \quad \mathbf{v} \sim \mathcal{N}(0,\, R_n)$$

  • $Q_n$ — process noise covariance (model uncertainty, disturbances)
  • $R_n$ — measurement noise covariance (sensor noise)

Observer (Kalman filter) dynamics:

$$\dot{\hat{\mathbf{x}}} = A\hat{\mathbf{x}} + Bu + L\left(\mathbf{y} - C\hat{\mathbf{x}}\right)$$

Kalman gain $L$ is found by solving the dual ARE:

$$A\Sigma + \Sigma A^T - \Sigma C^T R_n^{-1} C \Sigma + Q_n = 0$$

$$\boxed{L = \Sigma C^T R_n^{-1}}$$

where $\Sigma = \Sigma^T \succ 0$ is the steady-state error covariance.

The Kalman filter is the LQR of the dual problem — elegant symmetry.

5. LQG: The Combined Controller

The LQG controller separates by the Separation Principle:

$$\underbrace{\dot{\hat{\mathbf{x}}} = (A - LC)\hat{\mathbf{x}} + Bu + L\mathbf{y}}_{\text{Kalman Filter (Observer)}}$$

$$\underbrace{u = -K\hat{\mathbf{x}} + K\mathbf{x}_{\text{ref}}}_{\text{LQR State Feedback}}$$

Full closed-loop augmented system:

$$\frac{d}{dt}\begin{bmatrix} \mathbf{x} \\ \mathbf{e} \end{bmatrix} = \begin{bmatrix} A - BK & BK \\ 0 & A - LC \end{bmatrix} \begin{bmatrix} \mathbf{x} \\ \mathbf{e} \end{bmatrix} + \begin{bmatrix} BK \\ 0 \end{bmatrix}\mathbf{x}_{\text{ref}}$$

where $\mathbf{e} = \mathbf{x} - \hat{\mathbf{x}}$ is the estimation error.

Key result — Separation Principle:

$$\sigma(A - BK) \cup \sigma(A - LC)$$

The eigenvalues of the full system are exactly the union of the LQR poles and the Kalman filter poles — they do not interact. Design them independently.

6. Setpoint Tracking

For reference tracking to $\mathbf{x}_{\text{ref}} = \begin{bmatrix} p_{\text{ref}} \\ 0 \end{bmatrix}$:

Error state: $\tilde{\mathbf{x}} = \mathbf{x} - \mathbf{x}_{\text{ref}}$

Control law:

$$u = -K(\hat{\mathbf{x}} - \mathbf{x}_{\text{ref}}) = -K\hat{\tilde{\mathbf{x}}}$$

For zero steady-state error with a constant reference, an integrator on position error may be augmented:

$$\dot{\xi} = p_{\text{ref}} - p, \quad u = -K\hat{\mathbf{x}} + k_i \xi$$

This extends the state vector to $\begin{bmatrix} p & v & \xi \end{bmatrix}^T$ and the ARE is solved on the augmented system.

7. Design Summary

Component Equation Tunes
LQR cost $J = \int(\mathbf{x}^TQ\mathbf{x} + u^TRu)\,dt$ $Q, R$
LQR gain $K = R^{-1}B^TP$ Closed-loop speed
Kalman gain $L = \Sigma C^T R_n^{-1}$ Observer bandwidth
LQG control $u = -K\hat{\mathbf{x}} + K\mathbf{x}_{\text{ref}}$ Full system

8. Intuition

$$\underbrace{Q \uparrow}_{\text{track harder}} \Rightarrow K \uparrow \Rightarrow \text{faster, more aggressive control}$$

$$\underbrace{R \uparrow}_{\text{save energy}} \Rightarrow K \downarrow \Rightarrow \text{slower, gentler control}$$

$$\underbrace{Q_n \uparrow}_{\text{trust model less}} \Rightarrow L \uparrow \Rightarrow \text{trust measurements more}$$

$$\underbrace{R_n \uparrow}_{\text{noisy sensor}} \Rightarrow L \downarrow \Rightarrow \text{trust model more}$$

LQG is the optimal observer-based controller for linear systems with Gaussian noise — the gold standard before robust $\mathcal{H}_\infty$ methods entered the picture.
